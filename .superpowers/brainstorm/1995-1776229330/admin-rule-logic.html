A — Zakresy w prawdziwych jednostkach

Admin ustawia min/max w °C, indeksie UV (0–11), AQI (0–300), wilgotności (%), opadach (%). Reguła aktywuje się gdy aktualna pogoda mieści się w zakresie. Intuicyjne, zero normalizacji.

B — Gotowe presety warunków

Admin wybiera predefiniowane warunki ("Gorąco", "Zimno", "Wysokie UV", "Deszczowo", "Zła jakość powietrza"). Można łączyć wiele. Wartości progowe na sztywno wbudowane. Najprostsze w obsłudze.

C — Obecny system + lepsze etykiety

Suwaki 0–100 zostają, ale pokazują prawdziwe wartości przy aktualnej pozycji (np. "↑ tu: 28°C"). Mniejszy refactor, ale koncepcja "celu + tolerancji" nadal może być myląca dla kosmetologa.
