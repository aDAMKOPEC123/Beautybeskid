# Reel sprzedażowy — panel klienta BeskidStudio

Gotowy pionowy materiał promocyjny na Instagram Reels / Stories.

## Parametry

- format: 1080 × 1920 px (9:16)
- długość: 18 sekund
- płynność: 60 kl./s
- kodek: H.264 + AAC
- bezpieczne, krótkie komunikaty czytelne na telefonie
- autorski, subtelny ambient; można go zastąpić trendującym dźwiękiem w Instagramie

## Narracja

1. Twoja pielęgnacja. Zawsze pod ręką.
2. Wszystko w jednym miejscu.
3. Rezerwacja w kilku krokach.
4. Program lojalnościowy.
5. Spersonalizowany Beauty Plan.
6. Prywatny czat z salonem.
7. Zaproszenie na KosmetologWiktoriaCwik.pl.

## Ponowny render

```powershell
python render_reel.py
```

Plik wynikowy powstaje w katalogu `output/`.
